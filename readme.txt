Criar uma aplicação do Tipo Xamarin Forms para Android e iOS, com 3 telas em formato de 
Tabbed Page, que possibilite a consulta de CEPs, registro das pesquisas realizadas em banco de 
dados local e exibição dos locais (Pin) num mapa.

1) Instalação dos pacotes NuGet necessarios para execução do App; Newtonsoft.Json, PCLExt.FileStorage, sqlite-net-pcl, Xamarin.Essentials, Xamarin.Forms.Maps

2) Criação da pasta Model para modelo MVVM (Model, View, ViewModel).

3) Utilização do sqlite-net-pcl para criação do banco de dados ("person.db3").

4) Xamarin.Forms.Maps com API KEY do Google Developer.
